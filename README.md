# AI Supply Chain Operations Dashboard

AI-assisted operations dashboard for supply chain teams with issue tracking, recommendations, and audit logs.

## Goal

This project is a portfolio build focused on operational visibility, issue tracking, and human-in-the-loop AI workflows for supply chain teams.

It is designed to demonstrate practical full-stack skills for roles involving dashboards, internal tools, automation, and AI-assisted development.

## Planned Features

- Operations dashboard with key metrics
- Shipments, inventory, and issue tracking
- AI-generated summaries and recommended actions
- Human approval or rejection of AI suggestions
- Audit log for AI-assisted decisions
- Responsive interface for desktop and mobile

## Tech Stack

- Next.js
- React
- TypeScript
- API routes
- PostgreSQL / Supabase
- CSS modules or modern CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
